# Agent Instructions

System-wide instructions for all agent sessions are in `~/.agents/`.

Read `~/.agents/README.md` for the full directory index. At minimum, read
`~/.agents/github.md` at the start of every session since GitHub operations
are common across all projects.

## Directory layout

| File                    | Topic                                                   |
|-------------------------|---------------------------------------------------------|
| `~/.agents/README.md`   | Directory index and reading guide                       |
| `~/.agents/github.md`   | Which `gh` account to use for which repository owner    |
| `~/.agents/markdown.md` | Markdown style rules for all `.agents/` files           |
